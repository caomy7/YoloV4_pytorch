# YoloV4

Another implementation of YoloV4 in pytorch. In this repo we are gonna implement several basic concept of YoloV4, and then finally reveal result of yolov4.



- [ ] CSPDarknet;
- [ ] SPP;
- [ ] PANet;
- [ ] Mosiac augmentation;
- [ ] CutMix augmentation;
- [ ] CutOut augmentation;
- [ ] Generic algorithm searching hyper params;
- [ ] Inference result;
- [ ] Training process.
